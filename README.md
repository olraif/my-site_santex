# Сайт «Сантехник Кармаскалы»

Готовый адаптивный лендинг по макету: `index.html`, `styles.css`, `script.js`, изображения и базовые SEO-файлы.

## Что уже сделано

- Адаптивная вёрстка для компьютера и телефона.
- Быстрые CTA-кнопки: звонок, WhatsApp, ВКонтакте.
- SEO: title, description, Open Graph, canonical, robots.txt, sitemap.xml.
- Микроразметка Schema.org: LocalBusiness/Plumber и FAQPage.
- Семантические блоки: услуги, преимущества, зоны выезда, отзывы, FAQ, контакты.
- Оптимизированные изображения в WebP/JPG.

## Перед публикацией

1. Замените домен `https://santehnik-karmaskaly.ru/` в `index.html`, `robots.txt`, `sitemap.xml`, если домен будет другим.
2. При необходимости замените телефоны и ссылку ВКонтакте.
3. Загрузите папку на хостинг: Netlify Drop, Vercel, GitHub Pages, Timeweb, Beget или REG.RU.
4. После запуска добавьте сайт в Яндекс Вебмастер и Google Search Console.
5. Подключите Яндекс Метрику/Google Analytics, если нужна статистика заявок.

## Как открыть локально

Откройте файл `index.html` в браузере.


## Быстро показать заказчику

Самый простой способ:

1. Распакуйте архив.
2. Откройте Netlify Drop.
3. Перетащите туда всю папку `santehnik-karmaskaly`, а не отдельный файл.
4. После загрузки сервис выдаст временную ссылку вида `...netlify.app` — её можно отправить заказчику.

Важно: в корне загружаемой папки должен лежать `index.html`.
